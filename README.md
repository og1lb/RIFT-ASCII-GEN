# RIFT ASCII Gen

A fast and lightweight ASCII art generator that converts text into stylish ASCII designs.

## Features

- 🔤 Generate ASCII art from any text
- 🎨 Multiple font styles
- ⚡ Fast & Lightweight
- 💻 Cross-Platform
- 📋 Easy to Copy & Share
- 🔧 Simple and User-Friendly Interface
